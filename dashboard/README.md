# Dashboard Starter

This folder contains the optional Next.js dashboard task starter.

## Run Locally

From the `dashboard/` folder:

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

The app expects `papcorns.sqlite` to be at the repository root, one level above this folder.

## Where To Work

- Write the API/query in `app/api/subscription-revenue-breakdown/route.ts`.
- Add the chart in `app/page.tsx`.
